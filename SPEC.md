# TIC并购资讯智能采集与视频生成系统

## 1. 项目概述

- **项目名称**: TIC Intelligence Hub (检验检测认证资讯智能中枢)
- **核心功能**:
  1. 自动抓取TIC领域并购资讯、政策动态、头部公司新闻
  2. 基于抓取内容智能生成视频
- **目标用户**: TIC行业从业者、投资机构、分析师

## 2. 系统架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   爬虫引擎      │ ──► │   数据处理层     │ ──► │   视频生成引擎  │
│   (Crawler)     │     │   (Processor)    │     │   (Video Gen)   │
└─────────────────┘     └──────────────────┘     └─────────────────┘
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 ▼
                    ┌──────────────────────┐
                    │     Web 控制台       │
                    │   (React Dashboard)  │
                    └──────────────────────┘
```

## 3. 爬虫数据源

### 3.1 并购资讯
- 质量大号 (检测认证行业自媒体)
- 中国认证认可
- 检测机构的官网新闻

### 3.2 政策动态
- 国家市场监督管理总局
- 国家认监委
- 各省市市场监管局

### 3.3 头部公司动态
- 华测检测(SZ:300012)
-广电计量(SZ:002967)
- 国检集团(SH:603060)
- 中国汽研(SH:601965)
-谱尼测试(SZ:300887)

## 4. 数据模型

```javascript
// 资讯条目
{
  id: string,
  title: string,           // 标题
  summary: string,         // 摘要
  content: string,         // 完整内容
  source: string,         // 来源
  url: string,            // 原文链接
  publishedAt: Date,      // 发布时间
  category: 'ma' | 'policy' | 'company',  // 类别
  tags: string[],         // 标签
  sentiment: 'positive' | 'neutral' | 'negative',  // 情感
  videoGenerated: boolean, // 是否已生成视频
  createdAt: Date
}
```

## 5. 视频生成模块

### 5.1 视频模板
- 固定片头片尾
- 自动生成字幕
- AI配音(可选)
- 背景音乐

### 5.2 视频内容结构
```
[片头] → [今日要闻] → [政策解读] → [公司动态] → [行业趋势] → [片尾]
```

### 5.3 技术实现
- 使用 FFmpeg 进行视频合成
- 使用 PaddleOCR 生成字幕
- 使用 Web Speech API 或第三方TTS配音

## 6. 前端功能

### 6.1 资讯看板
- 实时显示爬取的资讯
- 支持筛选(类别/时间/来源)
- 关键词搜索
- 舆情分析展示

### 6.2 视频制作工坊
- 选择要制作成视频的资讯
- 选择视频模板
- 预览和导出

### 6.3 任务管理
- 爬虫任务调度
- 视频生成队列
- 执行日志

## 7. 验收标准

1. 成功爬取至少5个来源的TIC资讯
2. 资讯数据正确存储和展示
3. 能选择资讯生成视频
4. 生成的视频包含字幕和背景音乐
5. Web界面美观易用
